# Data Analysis AI Agent

## Overview

The Data Analysis AI Agent is an intelligent system that enables users to perform data analysis using natural language instead of writing code manually. The agent interprets user questions, generates appropriate Pandas operations, executes them on the provided dataset, and returns the results in a human-readable format.

The goal of this project is to bridge the gap between non-technical users and data analysis by automating common analytical tasks such as filtering, aggregation, statistical calculations, and data exploration.

---

## Features

* Natural Language to Pandas Query Generation
* Automated Data Analysis
* Dataset Exploration and Inspection
* LLM-Powered Query Understanding
* Real-Time Query Execution

---

## System Architecture

```text
User Query (Natural Language)
            │
            ▼
     Language Model
            │
            ▼
 Generate Pandas Code
            │
            ▼
 Code Validation Layer
            │
            ▼
 Execute Pandas Query
            │
            ▼
    Process Results
            │
            ▼
      User Output
```
